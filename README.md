# SynchroMed - Gestión Clínica Hospitalaria

## Módulo de Triaje Digital
**Descripción:** Clasificación automática de pacientes mediante lógica Java, priorizando la urgencia ("efecto embudo") y mitigando la saturación de atención en salud pública.

**Tecnologías:** 
- Java 17
- Maven
- Gestión de dependencias (Google Guava, Apache Commons Lang, JUnit 5)
- Persistencia de Datos con base Relacional (MySQL / JDBC)

## Reportes Clave Integrados:
1. Resumen de tiempos de espera por nivel de prioridad (Triaje).
2. Reporte de consumo de insumos vitales por procedimiento programado.
